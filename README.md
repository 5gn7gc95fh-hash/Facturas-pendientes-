# Facturas SRI

Programa web listo para GitHub Pages.

## Archivos incluidos

- `index.html`: programa principal.
- `manifest.json`: configuración para instalar en pantalla de inicio.
- `sw.js`: caché básico para abrir como app.
- `icon.svg`: icono de la app.

## Subir a GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos archivos en la raíz del repositorio.
3. Entra en Settings > Pages.
4. En Source elige Deploy from a branch.
5. Branch: main.
6. Folder: /root.
7. Guarda y espera el enlace.

## Uso

- Ingresa primero el producto.
- Luego los datos del cliente.
- El sistema guarda fecha y hora automáticamente.
- El IVA es 15%.
- Usa decimales con punto.
- Guarda registros en el navegador con localStorage.
